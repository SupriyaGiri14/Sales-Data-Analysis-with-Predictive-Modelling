# Sales-Data-Analysis-with-Predictive-Modelling
A Power BI sales analysis and forecasting project that explores sales trends, product and regional performance, and profitability. Interactive dashboards provide business insights, while Python-based predictive models forecast future sales to support inventory planning, resource allocation, and data-driven decision-making.

# Technologies Used

* Power BI – Interactive dashboards, data visualization, KPI tracking, and business insights

* Python – Data preprocessing, analysis, and predictive modeling

* Pandas – Data cleaning, transformation, feature engineering

* NumPy – Numerical computations

* Statsmodels – Time series forecasting (Holt-Winters, Exponential Smoothing)

* Kaggle Notebook – Python-based analysis and model development environment

* Microsoft Excel / CSV – Raw data storage and intermediate data handling

DAX (Power BI) – Measures, calculated columns, and advanced analytics

Time Series Analysis Techniques – Trend, seasonality, and forecasting models
